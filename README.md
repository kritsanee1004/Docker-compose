# Docker-compose

### Build  a micro-service :

```sh
$ docker-compose build
```

### Lunch service :

```sh
$ docker-compose up -d

```
### Stop service :

```sh
$ docker-compose down
```

### Access a running container :
```sh
$ docker-compose exec $SERVICE_NAME bash
```
